# Jatin Kola | Personal Portfolio

A cinematic, highly interactive, and professional portfolio website designed to showcase my work as a Full Stack Developer and UI/UX Designer. This project features a modern, premium aesthetic with dynamic animations and an immersive user experience.

![Portfolio Preview](/public/fullstack.png)

## ✨ Key Features

- **Cinematic Hero Section**: Implements a high-fidelity typewriter effect for greetings and professional titles, creating a seamless first impression.
- **Interactive Skills Universe**: A custom-built orbit system where skills rotate around a central hub. Clicking a skill node reveals detailed information with perfectly aligned connector lines.
- **Project Showcase**: A curated gallery of my work, featuring detailed descriptions, toolsets used, and key accomplishments.
- **Tools Grid**: A visual representation of the modern development and design tools I use daily.
- **Responsive Design**: Fully optimized for all screen sizes, ensuring a consistent and premium experience on desktop, tablet, and mobile.
- **Modern UI/UX**: Built with a focus on visual excellence, featuring glassmorphism, smooth gradients, and subtle micro-animations.

## 🛠️ Tech Stack

- **Core**: HTML5, Semantic UI
- **Logic**: Javascript (ES6+)
- **Styling**: Vanilla CSS3 (Custom design system)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Design Tools**: Figma, FigJam, Miro

## 📦 Getting Started

To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jatink16/personal-portfolio.git
   ```

2. **Navigate to the directory**:
   ```bash
   cd personal-portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
